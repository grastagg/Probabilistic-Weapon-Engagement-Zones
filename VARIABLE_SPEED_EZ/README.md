# VARIABLE_SPEED_EZ

This folder contains experiments for engagement-zone models with variable
pursuer speed profiles.

Core file:

- `variable_speed_ez.py` derives and visualizes engagement-zone conditions for
  piecewise-constant pursuer speed schedules, including safe-heading cone
  constructions and related plotting helpers.

This folder is currently a compact research script rather than a full module
tree, with the model derivation and visualization code living in the same file.
