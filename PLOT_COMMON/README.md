# `PLOT_COMMON`

Shared plotting helpers used by multiple engagement-zone studies.

## Files

- `draw_airplanes.py`: reusable airplane glyphs plus a small standalone demo.
- `draw_mahalanobis.py`: covariance-based contour overlay for pursuer uncertainty.
- `curlyBrace.py`: vendored matplotlib curly-brace annotation helper.

## Usage Notes

- These helpers are imported by several folders in the repository.
- `draw_airplanes.py` contains both reusable plotting helpers and a standalone demo entry point.
- `curlyBrace.py` is third-party utility code used for figure annotations.
